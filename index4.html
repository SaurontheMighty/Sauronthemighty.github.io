H to O

A project I built with some friends for a hackathon.

...

So far I've attended 2 online hackathons organized by MLH and I've built Flutter projects for both!

The first project is called H to O and the main idea behind this was to create an app that would allow you to create a shopping list of the items you need and then show you the store that was closest to you and had the items you wanted. I used the Google Maps API to create the map and add the markers which showed each location. When you tap the description it takes you to a Google Maps page with the directions to the store. I also pulled data from our Firestore database and updated the maps accordingly. Any change in the inventory of the store can be quickly reflected in the map. This was a really fun project for me because I got to learn how to use the Google Maps API which I hadn't used before.



The other project is called FoodConnect and it's designed to connect donors to the Foodbanks. The idea was that a donor could add the items they have and the app would connect them to a Foodbank that needed those items.
